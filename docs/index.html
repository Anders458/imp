imp

AI git CLI for commit messages, history, changelogs, and PRs.

View on GitHub Install from PyPI

What it does

imp rewrites a dirty working tree into clean Conventional Commits and ships them with a real changelog. One command, every time.

Install

pip install imp-git
imp config
imp doctor

Works with the Claude CLI or local Ollama models. No API key required for Ollama.

Commands

CommandPurpose
imp commitGenerate a Conventional Commit from your diff
imp splitGroup a dirty working tree into logical commits
imp tidyReword, squash, and drop past commits with AI
imp prOpen a pull request with AI title and body
imp releaseSquash, changelog, tag, and push a release
imp resolveAI-assisted merge conflict resolution
imp reviewAI code review on a branch or commit range

Why imp

  • git doesn't write commit messages. You do. Badly, at 2am.
  • git doesn't group changes. You end up with wip, fix stuff, more fixes.
  • git doesn't enforce Conventional Commits. imp validates before anything lands.
  • git doesn't know what changed. imp reads the diff and writes the message.
